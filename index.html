Welcome to SKII Programming Language

A skiing-inspired language for all coding levels!

Introduction

SKII is a fun programming language designed to help you progress from beginner to advanced coding concepts using ski-related terminology. The language is divided into three levels: GREEN, BLUE, and BLACK DIAMOND.

GREEN Level Grammar

The GREEN level is for beginners, where you start learning basic commands like printing and assigning variables.

Example:

SKII GREEN { lodge speed as 10; snow(traverse speed to 20); }

In this example:

  • lodge: declares a variable.
  • snow: prints the result.
  • traverse: adds two values.

BLUE Level Grammar

The BLUE level introduces more advanced features like loops, conditionals, and lists.

Example:

SKII BLUE { slope(descend base 0 from summit 100) { snow(lodge speed as 10); } apres step 0 to summit 100 by 10 { snow(traverse speed to step); } }

In this example:

  • slope: a while loop.
  • apres: a for loop with a start, end, and step.
  • snow: prints the result.

BLACK DIAMOND Level Grammar

The BLACK DIAMOND level is for advanced programmers, with features like classes, recursion, and error handling.

Example:

SKII BLACK DIAMOND { steez Snowboard { ~frontside speed() { snow(lodge speed as 20); } } }

In this example:

  • steez: defines a class.
  • ~frontside: defines a public method.
